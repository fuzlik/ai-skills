# Research Brief

## Цель

Дать решение/ориентир, а не дамп ссылок.

## Шаги

1. Уточни критерий решения (если критично и неясно — один короткий вопрос).
2. Собери факты из кода, доков, веб-поиска — по приоритету задачи.
3. Отдели факт / вывод / неизвестное.
4. Выдай бриф по шаблону.

## Шаблон

```markdown
## Verdict
1–2 предложения: рекомендация.

## Options
| Option | Pros | Cons | When |
|--------|------|------|------|
| A | | | |
| B | | | |

## Key facts
- факт (+ источник)

## Unknowns / risks
- …

## Next step
Конкретное действие на 15–60 минут.
```

## Правила

- Сначала verdict.
- Не больше 5–7 ключевых фактов.
- Указывать источники для спорных утверждений.
- Не раздувать в эссе.
