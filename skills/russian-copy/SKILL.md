# Russian Copy

## Тон по умолчанию

- Прямой, деловой, без канцелярита.
- Короткие предложения.
- «Ты» в продуктовых UI — только если так принято в продукте; иначе нейтрально.

## Правила

1. Сначала смысл, потом красота.
2. Одна мысль — один абзац/буллет.
3. Избегать: «осуществлять», «в рамках», «данный», «является».
4. Глаголы действия в UI: «Сохранить», «Отправить», не «Произвести сохранение».
5. Англицизмы — только устоявшиеся (API, PR, commit) или принятые в команде.

## Форматы

**UI-строка:** ≤ 60 символов где возможно; без точки в конце кнопки.

**Документ:** заголовок → суть → шаги → результат.

**Письмо/сообщение:** цель в первом предложении; дальше детали; явный CTA.

## Вывод

Дай 1 основной вариант. Альтернативы — только по просьбе (макс. 2).
